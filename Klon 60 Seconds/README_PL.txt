60 Seconds Klon - Gra w Pygame/Python

Jest to klon gry "60 Seconds" autorstwa Robot Gentleman.

Instrukcja uruchomienia:
1. Upewnij się, że masz zainstalowanego Pythona (zalecana wersja Python 3.6 lub nowsza)
2. Zainstaluj wymagane biblioteki: pip install pygame numpy
3. Uruchom grę: python standalone_game.py

Sterowanie:
- WASD lub strzałki do poruszania się
- SPACJA, aby podnieść/upuścić przedmioty

Gra ma dwie fazy:
1. Zbieranie: Masz 60 sekund na zebranie zapasów z domu
2. Przetrwanie: Podejmuj codzienne decyzje, aby utrzymać swoją rodzinę przy życiu

Miłej gry!
